## What is this?

https://github.com/user-attachments/assets/fb64dc93-37e7-421f-bc5d-89a8ef43b7c8

This is a UserScript. The initial goal was to replicate the built-in word selection toolbar functionality found in modified browsers like Qihoo 360 Browser and Microsoft Edge, within native browsers like Chrome and Firefox. Since I didn't want to have to press the right mouse button or Ctrl+C shortcut every time I wanted to copy text, I used AI to write a simple script that allows for quick copying after selecting text. Later, as my ideas grew, I eventually pieced together this more feature-rich UserScript.

## Basic Features

* The user interface mimics the style of Apple Liquid Glass

* Support for multiple languages (via machine translation)

* Focus on low memory usage

* Event-driven, avoiding CPU-intensive code such as polling

* By default, no intrusive modifications are made to the webpage, and the script runs in a separate context to avoid polluting the webpage

* 100% offline operation (the script auto-update function is implemented by the script manager through regular reading of script metadata segments, and the script itself does not require internet connection)

* No obfuscated code, no commercial or malicious behavior
